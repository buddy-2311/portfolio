## How to Use
1. Open `index.html` in your browser.
2. The profile photo is already added at `assets/profile.jpg`.
3. The resume is already added at `assets/Hinal_Suthar_Resume.pdf`.
4. The contact form is connected to FormSubmit and sends messages to `hinalsuthar2311@gmail.com`.

## Notes
For FormSubmit, the first live submission may ask the email owner to verify/activate the email address. After activation, messages are delivered directly to the inbox.
